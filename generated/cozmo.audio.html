cozmo.audio¶

Audio related classes, functions, events and values.

Classes

AudioEvents The possible values for an AudioEvent.
class cozmo.audio.AudioEvents¶

The possible values for an AudioEvent. Pass one of these event objects to robot.play_audio() to play the corresponding sound clip. Example: robot.play_audio(cozmo.audio.AudioEvents.MusicFunLoop)
